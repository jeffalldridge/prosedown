# Why Plain Text

The case for writing in Markdown is not about technology. It's about ownership.

When you write in a proprietary format, your words live inside someone else's software. When you write in plain text, your words are yours — readable on any device, editable in any tool, preserved for decades.[^1]

> "The best writing tool is the one that disappears."

This is why every serious writing workflow eventually converges on plain text with structured metadata.

---

The shift is already happening. Obsidian, iA Writer, and VS Code have become the writing environments of choice for a growing number of authors.[^2]

[^1]: Plain text files from the 1970s are still readable today. Try opening a WordPerfect file from 1995.

[^2]: See the annual Stack Overflow developer survey for text editor trends, and the Obsidian community census for writing-specific usage.
