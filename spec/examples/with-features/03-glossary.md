# Glossary

CommonMark
: An open standard for Markdown syntax, maintained as a formal specification. ProseDown uses CommonMark as its Markdown base.

EPUB
: Electronic Publication. An open standard for digital books, maintained by the W3C. ProseDown compiles to EPUB 3.3.

Frontmatter
: The opening pages of a book that come before the main content — copyright, dedication, foreword, table of contents.

Backmatter
: The closing pages of a book — appendix, glossary, bibliography, about the author.

YAML
: A human-readable data format used for metadata. ProseDown uses YAML frontmatter (the `---` delimited block at the top of each file) for book and chapter metadata.

Slug
: A URL-safe version of a string, used for filenames and anchor IDs. Created by lowercasing, replacing spaces with hyphens, and removing special characters.
